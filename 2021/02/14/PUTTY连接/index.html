ssh安装 启动

(一般ssh都不是自带的，需要自己安装)

  • sudo apt-get update
  • sudo apt-get install openssh-server

若显示无法等待缓存锁：无法获得锁

sudo rm /var/lib/dpkg/lock-frontend

sudo rm /var/cahe/apt/archives/lock

sudo rm /var/lib/dpkg/lock

  • 查看ssh是否启动

    ​ sudo ps -e | grep ssh

    • 有**sshd**就是启动了

    • 若没有启动

      sudo service ssh start
